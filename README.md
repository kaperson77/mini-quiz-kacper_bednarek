# Mini Quiz - Autor: Kacper Bednarek

## Opis projektu
Quiz w Android Studio

## Funkcje
- Losowanie pytań z tablicy wielowymiarowej.
- Licznik punktów.
- Możliwość resetowania quizu.
- Interfejs zbudowany na LinearLayout.

## Instrukcja uruchomienia
1.Uruchom.
2. Poczekaj na zakończenie budowania projektu przez Gradle.
3. Uruchom aplikację na emulatorze lub fizycznym urządzeniu (Shift + F10).
